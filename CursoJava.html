¿Qué es Java?

Se basa en una programación orientada a objetos, permite ejecutar un mismo programa en múltiples sistemas operativos y hace posible ejecutar el código en sistemas remotos de manera segura. Su ámbito de aplicación es tan amplio que Java se utiliza tanto en móviles como en electrodomésticos.
La importancia de Java radica en la posibilidad de crear o diseñar software y poder ser ejecutado en diferentes plataformas donde se requiere, sin la necesidad de estar modificándolo.
